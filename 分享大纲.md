### **0. 开场：为什么要换掉你手里的铲子？（5分钟）**

* **痛点唤醒：**
  * 大家有没有这种经历：PPT 做好了，老板说“数据变了，把 Excel 重新导一遍”，你得重新截图、重新调位置、重新对齐？
  * **传统模式（像画油画）：** 画好了就干了，改一笔得刮掉重画。
  * **今天的新模式（像搭乐高）：** 数据是积木，逻辑是说明书。数据变了？换块积木就行，模型自动调整。
* **主角登场：**
  * **Gemini (谷歌)：** 它的脑容量极大，能一次性读完几十份报告，负责把杂乱的信息变成**“标准积木”**。
  * **Trae (字节)：** 它是一个能听懂人话的建造台，负责把积木搭成我们想要的**“房子” (PPT)**。

### **A. Gemini：首席建筑师——从“一锅粥”到“标准件”（20分钟）**

> **讲师话术：** “别把 Gemini 当聊天机器人，把它当成一台**‘信息提炼机’**。丢进去的是矿石，出来的是金条。”

#### **1. 数据“暴风吸入”与边界认知（10分钟）**

* **场景代入：**
  * 你手头有：一份去年的 PDF 总结报告、三张不同格式的 Excel 销售表、一段老板开会的录音文字稿。
  * **操作演示：**
    * **一键投喂：** 别一个个打开看，直接全选拖进 Gemini。
    * **跨界提问：** “结合 PDF 里的市场策略和 Excel 里的实际销量，告诉我哪里执行得不到位？”（强调：它能跨文件建立联系，这是人脑最累的地方）。
* **避坑指南（关键点）：**
  * **Gemini 的短板：** 它不是计算器。如果你问它“12345 x 67890 等于多少”，它可能一本正经地胡说八道。
  * **正确用法：** 让它做**定性分析**（找趋势、找异常、写总结），涉及精确计算的，让它写公式，不要让它直接算结果。

#### **2. 核心动作：制造"标准积木" (10分钟)**

* **这是本场培训最重要的一分钟：**
  * 为了让下一个工具（Trae）能直接干活，我们不能让 Gemini 输出"小作文"，必须让它输出**"机器能读懂的清单"**。
  * **非技术解释：** 想象你要搬家。
    * 普通输出："把东西都装箱。"（Trae 听不懂，会乱装）
    * **CSV 格式输出：** "1号箱：书，2号箱：衣服"（这就是 CSV，一种简单的数据清单）。
* **实操 Prompt（口令）：**
  * "分析完这些数据后，请把'地区'、'月份'、'销售额'整理成 **CSV 格式**。第一行是列名，后续行是数据，不要任何多余的废话。"
  * *结果：* 屏幕上出现一段表格格式的文本。告诉学员：**"看不懂没关系，复制它。这就是我们炼出来的金条。"**
* **进阶提示（选讲）：**
  * Gemini 不仅能清洗数据，还能**直接计算分析指标**：
    * "同时帮我计算完成率（实际÷目标×100%）"
    * "计算同比增长率（今年-去年）÷去年×100%"
  * 这样输出的 CSV 直接包含分析结果，省去后续计算步骤

### **A.5 真实场景实战示例（选讲，10分钟）**

> **讲师话术：** "理论都懂了，我们来看看实际工作中怎么用。这些都是真实学员的需求场景。"

**场景一：人员信息表分析（组织架构分析）**

**需求背景：**
- 数据来源：Excel 人员信息表（包含部门、职级、入职时间、年龄、学历等）
- 分析目标：分析组织组成结构，输出可视化报告

**Gemini 操作流程：**
```
第一步：上传 Excel 文件到 Gemini
第二步：输入 Prompt

"请分析这份人员信息表，按以下维度整理成 CSV 格式：

1. 部门分布：部门名称,人数,占比
2. 职级结构：职级名称,人数,占比
3. 年龄分布：年龄段,人数,占比
4. 学历分布：学历,人数,占比

每个维度输出一个独立的 CSV 表格。"
```

**Trae 可视化：**
```
"基于以下 4 组 CSV 数据，创建组织分析仪表盘：

[粘贴部门分布 CSV]
[粘贴职级结构 CSV]
[粘贴年龄分布 CSV]
[粘贴学历分布 CSV]

要求：
- 2x2 网格布局，每个维度一个饼图
- 左上：部门分布（配色用蓝色系）
- 右上：职级结构（配色用绿色系）
- 左下：年龄分布（配色用紫色系）
- 右下：学历分布（配色用橙色系）
- 图表标题："公司组织结构分析看板"
```

**预期成果：** 一页展示公司组织全貌的数据看板

**场景二：广告投放数据分析（运营效果分析）**

**需求背景：**
- 数据来源：Excel 广告投放数据（渠道、投放金额、曝光量、点击量、转化量）
- 分析目标：分析趋势规律，给出数据支撑的结论

**Gemini 高级分析：**
```
"请分析这份广告投放数据，完成以下任务：

1. 数据清洗：
   - 计算每个渠道的点击率（点击÷曝光×100%）
   - 计算转化率（转化÷点击×100%）
   - 计算 ROI（转化价值÷投放金额×100%）

2. 趋势分析：
   - 识别表现最好的3个渠道
   - 识别需要优化的渠道（ROI<100%）
   - 找出异常数据点（突然下降的转化率）

3. 输出 CSV 格式：
   渠道,投放金额,曝光量,点击量,转化量,点击率,转化率,ROI,状态

状态标注：优秀（ROI>150%）、良好（ROI 100-150%）、待优化（ROI<100%）"
```

**Gemini 生成结论：**
```
"基于上述分析数据，请以运营专家身份给出：

1. 核心发现（Top 3）：哪些渠道值得加大投入？
2. 风险提示：哪些渠道需要暂停或调整？
3. 优化建议：具体的行动方案（渠道、预算、创意方向）

输出格式：
### 核心发现
1. [发现1 + 数据支撑]
...

### 风险提示
- ⚠️ [风险1 + 影响分析]
...

### 优化建议
- 🔴 高优先级：[具体措施 + 预期效果]
...
```

**Trae 完整演示：**
```
"创建广告投放分析PPT，包含3页：

第1页：数据总览表格
- 表格展示所有渠道数据
- ROI>150% 的行背景显示绿色
- ROI<100% 的行背景显示红色

第2页：趋势分析图表
- 双Y轴图表：柱状图显示投放金额，折线图显示ROI
- 在ROI=100%处画红色基准线

第3页：AI 洞察与建议
[粘贴 Gemini 生成的结论]
- 卡片式布局，优先级用颜色区分
```

**预期成果：** 3页完整的投放效果分析报告

### **B. Trae：智能3D打印机——说出想法，即刻呈现（30分钟）**

> **讲师话术：** “以前做 PPT 是‘排版一小时，内容五分钟’。现在用 Trae，我们只负责指挥，脏活累活它全包。”

### **讲师备课Tips（防翻车指南）：**

1. **CSV 格式说明：** 不要过度解释技术细节。就说CSV是**"Excel的纯文本版本"**，机器更容易理解。如果学员问"为什么不直接用Excel"，回答："Excel包含样式、公式等复杂内容，CSV只有纯数据，AI处理更稳定。"

2. **Trae 演示准备：** 现在的 AI 有时会"抽风"（比如图表加载不出来）。
   * *锦囊：* 准备一段**备用代码**。如果现场 Trae 生成的图表是空的，你就说："有时候 AI 也会忘事，我们提醒它一下"，然后把你准备好的正确代码贴进去让它"参考"。这反而显得你很懂如何驾驭 AI。

3. **关于"HTML PPT"：** 学员可能会问"能不能转成 PPTX 格式？"
   * *回答：* "可以转 PDF，但那样就失去了动画交互的灵魂。我们今天学的是**未来的汇报方式**，不是为了迁就过去的工具。"

4. **进阶内容时间分配：**
   * 基础学员：重点演示基础柱状图，进阶内容只提一下，引导课后学习
   * 技术背景学员：可以现场演示完成率分析图表，展示多维度能力
   * 如果有人问多维度分析，直接说："这个在课前手册第3.3节有详细讲解，包含完整的Prompt模板，课后可以深入学习。"

5. **素材准备清单：**
   * ✅ 备用CSV数据（已验证可用）
   * ✅ 备用HTML代码（完整可运行版本）
   * ✅ 课前手册PDF（分享给学员）
   * ✅ Prompt模板速查卡（打印或投屏）
