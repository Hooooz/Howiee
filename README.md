# AI工具汇报引擎演示

一个基于Reveal.js的AI工具使用教学演示文稿,展示如何使用Gemini和Trae进行数据分析和可视化。

## 🎯 特性

- ✅ 21页完整演示内容
- ✅ 4个真实业务场景案例
- ✅ 支持页面内滚动查看完整内容
- ✅ 集成ECharts数据可视化
- ✅ 响应式设计,支持16:9演示模式

## 📊 包含场景

1. **场景一**: 组织架构分析 - 人员信息表转组织结构看板
2. **场景二**: 广告投放分析 - ROI分析 + AI洞察
3. **场景三**: 业绩数据看板 - 多维度业绩指标展示
4. **场景四**: HR管理看板 - 人力资源管理驾驶舱

## 🚀 在线预览

打开 `demo-v3-refined.html` 即可查看演示。

### 操作说明

- **左右箭头**: 切换页面
- **鼠标滚轮**: 在当前页面内滚动查看完整内容
- **ESC键**: 查看演示概览
- **F键**: 全屏模式

## 📝 技术栈

- Reveal.js 4.5.0
- ECharts 5.4.3
- Google Fonts (Noto Sans SC + JetBrains Mono)
