## 🚀 快速开始

### 方式一：直接查看 Demo（推荐）

1. **打开 HTML 文件**
   ```bash
   # 方法1：双击文件
   直接双击 demo-enhanced.html

   # 方法2：使用浏览器打开
   拖拽 demo-enhanced.html 到 Chrome/Edge/Safari
   ```

2. **浏览器中查看**
   - 使用 ← → 方向键切换页面
   - 使用 ESC 键查看全部页面缩略图
   - 按 F 键进入全屏模式
   - 按 S 键打开演讲者视图

3. **主要功能**
   - ✅ 霓虹发光标题效果
   - ✅ 实时数据可视化图表（ECharts）
   - ✅ 平滑页面切换动画
   - ✅ 底部自动进度条
   - ✅ 鼠标悬停数据提示
   - ✅ 响应式布局（支持不同屏幕）

### 方式二：本地服务器运行（可选）

如果遇到跨域问题或需要分享链接：

```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000

# Node.js (需要先安装 http-server)
npx http-server -p 8000
```

然后访问：http://localhost:8000/demo-enhanced.html

## 🎨 自定义修改指南

### 修改配色方案

在 `demo-enhanced.html` 的 `<style>` 标签中找到：

```css
:root {
    --glow-orange: #ff6600;      /* 橙色强调色 */
    --neon-blue: #00f3ff;        /* 霓虹蓝 */
    --cyber-purple: #a855f7;     /* 赛博紫 */
    --bg-dark: #1a1a1a;          /* 深色背景 */
    --surface: #2b2b2b;          /* 卡片背景 */
    --text-primary: #ffffff;     /* 主文字色 */
    --text-secondary: #999999;   /* 次要文字色 */
}
```

修改这些颜色值即可全局改变配色。

### 修改数据内容

找到 JavaScript 部分的 `salesDataJSON`：

```javascript
const salesDataJSON = {
    "data": [
        {"Region": "华东区", "Month": "7月", "Sales": 100},
        // 修改这里的数据
    ]
};
```

替换为你自己的 JSON 数据即可。

### 添加新页面

在 `<div class="slides">` 中添加新的 `<section>`：

```html
<section data-background-color="#1a1a1a">
    <h2>新页面标题</h2>
    <p>页面内容...</p>
</section>
```

### 修改字体

在 `<head>` 中修改 Google Fonts 引用：

```html
<link href="https://fonts.googleapis.com/css2?family=你的字体名&display=swap" rel="stylesheet">
```

然后在 CSS 中应用：

```css
.reveal {
    font-family: '你的字体名', sans-serif;
}
```

## 💡 最佳实践

### 演示技巧

1. **提前测试**
   - 在实际投影设备上测试显示效果
   - 测试所有交互功能是否正常
   - 准备离线版本（下载所有 CDN 资源）

2. **演讲节奏**
   - 每页停留 1-2 分钟
   - 重点页面（图表页）停留 3-5 分钟
   - 实操演示控制在 15 分钟内

3. **互动设计**
   - 第 2 页后问观众："大家有这些痛点吗？"
   - 第 7 页演示图表交互："大家看这里..."
   - 最后一页："谁想试试？"

### 数据准备

1. **JSON 数据规范**
   ```json
   {
     "data": [
       {
         "字段1": "值1",
         "字段2": "值2",
         "字段3": 数值
       }
     ]
   }
   ```

2. **数据清洗 Prompt 模板**
   ```
   请分析以下数据：[粘贴原始数据]

   要求：
   1. 输出标准 JSON 格式
   2. 字段包含：[列出字段名]
   3. 数值类型为数字（不要加引号）
   4. 使用数组格式方便遍历
   ```

3. **常见数据类型**
   - 销售数据：地区、时间、金额
   - 用户数据：渠道、日期、数量
   - 业绩数据：部门、指标、完成度

### 样式优化

1. **保持简洁**
   - 每页最多 3-5 个要点
   - 避免文字过多
   - 使用图标和图表代替文字

2. **色彩搭配**
   - 主色不超过 3 种
   - 保持足够对比度
   - 避免过于鲜艳的颜色

3. **动画使用**
   - 仅在必要时使用动画
   - 动画时长控制在 0.3-0.5 秒
   - 避免过于复杂的动画效果

## 🔧 故障排除

### 问题 1：图表不显示

**可能原因**：
- ECharts CDN 未加载成功
- JSON 数据格式错误
- JavaScript 代码执行出错

**解决方案**：
1. 打开浏览器开发者工具（F12）
2. 查看 Console 是否有错误信息
3. 检查 Network 标签，确认 CDN 加载成功
4. 验证 JSON 数据格式是否正确

### 问题 2：页面切换不流畅

**可能原因**：
- 浏览器性能问题
- 图片或资源过大
- 动画效果过多

**解决方案**：
1. 使用 Chrome 或 Edge 浏览器
2. 关闭其他占用资源的程序
3. 减少动画效果或降低动画复杂度

### 问题 3：字体显示异常

**可能原因**：
- Google Fonts 加载失败
- 网络问题
- 浏览器不支持该字体

**解决方案**：
1. 检查网络连接
2. 使用备用字体（sans-serif）
3. 下载字体到本地引用

### 问题 4：移动端显示问题

**可能原因**：
- 响应式设计不完善
- 移动端浏览器兼容性

**解决方案**：
1. 使用桌面端进行演示
2. 调整 viewport 设置
3. 简化移动端显示内容
