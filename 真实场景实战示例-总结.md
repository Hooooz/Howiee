## 📊 场景详细说明

### 场景一：组织架构分析（入门级）

**适用对象：** HR、行政、管理层

**核心价值：**
- 一键将人员名单转化为可视化组织报告
- 多维度透视组织结构
- 快速发现人员结构问题（如：某部门学历分布不合理）

**Gemini 操作：**
```
输入：上传 Excel 人员信息表
输出：4个维度的 CSV（部门、职级、年龄、学历）
```

**Trae 操作：**
```
输入：4个 CSV 数据
输出：2x2 网格布局，每个维度一个饼图
效果：一页展示公司组织全貌
```

**预计耗时：** 10分钟

---

### 场景二：广告投放分析（中级 + AI洞察）

**适用对象：** 运营、市场、投放优化师

**核心价值：**
- 自动计算 ROI、转化率等关键指标
- AI 自动生成优化建议（不仅展示数据，还给出结论）
- 识别异常渠道，辅助决策

**Gemini 双重任务：**
1. **数据清洗 + 计算：** 输出包含点击率、转化率、ROI 的完整 CSV
2. **生成洞察：** 以运营专家身份给出核心发现、风险提示、优化建议

**Trae 多页演示：**
- 第1页：彩色表格（ROI 好坏用颜色区分）
- 第2页：双Y轴趋势图（投放金额 + ROI）
- 第3页：AI 洞察卡片（核心发现/风险/建议）

**预计耗时：** 15分钟

**进阶亮点：** 展示 AI 不仅能处理数据，还能**给出专业结论**

---

### 场景三：业绩数据看板（高级 + 多维度）

**适用对象：** 销售管理、业务分析、高管汇报

**核心价值：**
- 最常见的业务场景（业绩汇报）
- 展示多维度分析能力（完成率、同比、环比）
- 一屏展示所有关键指标，像汽车仪表盘

**Gemini 多维度输出：**
```
输入：包含产品/事业线/销售/团队/月份/目标/实际的原始表
输出：4个维度的独立 CSV
  - 产品维度汇总
  - 事业线维度汇总
  - 销售个人明细
  - 团队汇总
每个 CSV 都包含：完成率、达标状态、同比、环比
```

**Trae 4象限看板：**
```
左上：产品业绩对比（柱状图 + 折线图 + 达标线）
右上：事业线趋势（堆叠面积图 + 趋势预测）
左下：TOP 销售排行（横向柱状图 + 金牌图标）
右下：团队完成率（多个圆形仪表盘 + 颜色区分）
```

**预计耗时：** 20分钟

**讲师重点演示场景：** 最实用、最常见、最能体现工具价值

---

### 场景四：HR 管理看板（综合应用）

**适用对象：** HRBP、人力资源总监

**核心价值：**
- 人力资源管理驾驶舱
- 实时监控人员状态 + 预警机制
- 多维度人效分析

**Gemini 数据整合：**
```
输入：包含在职状态/部门/产出/入离职等多维度的人员数据
输出：4个维度的 CSV
  - 在职状态分布
  - 部门人效对比
  - 流动率趋势分析
  - 预警清单（即将到期、未晋升）
```

**Trae 管理看板：**
```
左上：人员状态总览（KPI 卡片 + 环形图）
右上：部门人效对比（雷达图 + 气泡图）
左下：流动率趋势（双轴图 + 预警线）
右下：预警清单（表格 + 颜色标注 + 可展开详情）
```

**预计耗时：** 15分钟

## 🎓 教学策略建议

### 时间分配方案

**60分钟课程：**
- 基础理论（A部分）：20分钟
- 真实场景演示（A.5）：10分钟（选场景三）
- Trae 实操（B部分）：25分钟
- 答疑（C部分）：5分钟

**45分钟课程：**
- 基础理论：15分钟
- Trae 实操（含场景三穿插讲解）：25分钟
- 答疑：5分钟

### 场景选择策略

| 学员背景 | 推荐场景 | 理由 |
|---------|---------|------|
| 销售/业务团队 | 场景三 | 最常见，业绩汇报是刚需 |
| 运营/市场团队 | 场景二 | 展示 AI 洞察能力，提供决策依据 |
| HR/行政团队 | 场景四 | 直接解决人员管理痛点 |
| 混合背景学员 | 场景三 + 场景二（简化版） | 覆盖最广泛的需求 |

### 演示技巧

**场景一（组织架构）：**
- **适合开场热身：** 简单、直观、快速出效果
- **演示重点：** 强调"上传Excel → 10分钟出看板"的效率
- **话术：** "大家看，人事部门每次做组织报告要花半天，现在10分钟搞定。"

**场景二（广告投放）：**
- **适合展示 AI 思维：** 不仅处理数据，还给结论
- **演示重点：** Gemini 生成的洞察报告（核心发现、风险、建议）
- **话术：** "注意看这里，AI不仅告诉你'哪个渠道ROI高'，还告诉你'为什么高、怎么复制'。这才是AI的价值。"

**场景三（业绩看板）：** ⭐ **重点推荐**
- **适合全流程演示：** 最复杂、最实用、最能展示多维度分析
- **演示重点：** 4象限看板，一屏展示所有维度
- **话术：** "传统PPT要做10页图表，我们一页搞定。老板看报告，就像开车看仪表盘，所有关键指标一目了然。"

**场景四（HR 看板）：**
- **适合深度演示：** 展示预警机制、颜色编码等高级功能
- **演示重点：** 预警清单的实用性
- **话术：** "人力资源管理最怕的是什么？试用期到期了没人提醒、合同到期了才发现。这个看板自动标红预警。"

## 💡 讲师话术模板

### 开场引入（A.5章节开始时）

```
"理论都懂了，我们来看看实际工作中怎么用。

刚才我收集了真实学员的需求，发现大家的问题无非这几类：
- HR 要分析人员结构
- 运营要分析广告效果
- 销售要做业绩汇报
- 管理层要看数据看板

今天我给大家准备了4个真实场景的完整流程，
看完你就知道，自己的工作怎么套用这套方法。"
```

### 场景演示过程中

```
[演示场景三时]

"大家注意看，这个业绩看板演示的是最常见的销售汇报场景。

Gemini 做了什么？
- 把混乱的 Excel 表整理成4个维度
- 自动计算完成率、同比、环比
- 每个维度一个独立的 CSV 文件

Trae 做了什么？
- 把4个 CSV 变成4个象限的图表
- 左上角看产品，右上角看事业线，左下角看销售，右下角看团队
- 一个页面，4个视角，所有维度一览无余

传统做法要多久？至少2小时。
现在多久？20分钟。

而且，数据变了怎么办？传统方法要重做。
现在呢？只需要更新源数据，刷新页面，图表自动更新。"
```

### 总结升华（A.5章节结束时）

```
"刚才这4个场景，大家看到了吗？

无论是人员分析、广告投放、业绩看板，还是HR管理，
核心逻辑都一样：

第一步：Gemini 数据清洗
第二步：Gemini 多维计算（完成率、同比、环比）
第三步：Trae 可视化呈现（图表、看板、洞察）

你们回去之后，把自己的需求套进这个流程，
90%的汇报问题都能解决。

剩下10%解决不了的，要么是数据太复杂，
要么是需求太个性化，那就来找我单独聊。"
```

## 🎯 预期学习成果

学员完成这4个场景演示后，应该能够：

- ✅ 能主动尝试组合多个场景（如：业绩看板 + AI 洞察）
- ✅ 能根据业务需求定制新的分析维度
