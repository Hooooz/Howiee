## 任务一：Gemini 数据清洗

### 🎯 演示目标
展示 Gemini 如何将杂乱的文本数据转换为标准 JSON 格式

### 📋 步骤 1：准备模拟数据

**场景设定（对观众说）：**
```
假设你是销售经理，手里有一堆乱七八糟的销售记录，
需要分析 Q3 的业绩并做成可视化图表。
```

**投喂素材（复制粘贴到 Gemini）：**

```text
这里有几段销售记录，请帮我整理：

1. 邮件内容：华东区7月份卖了咖啡机500台，单价2000元；但是8月份因为缺货，只卖了300台。9月份回升到600台。

2. 会议纪要：华南区那边，7月是40万销售额，8月是45万，9月猛增到80万（因为做了促销）。

3. 客服反馈：华北区一直很稳，每个月都是固定50万销售额。
```

### 📋 步骤 2：关键 Prompt

**复制粘贴到 Gemini：**

```text
请分析上述数据，计算出三个大区7-9月的每月销售额（单位：万元）。

关键要求：
1. 不要给我写文本分析
2. 请直接输出为标准的 JSON 格式数据
3. 结构包含：Region（区域）, Month（月份）, Sales（销售额）
4. 数组格式，方便程序读取

示例格式：
{
  "data": [
    {"Region": "华东区", "Month": "7月", "Sales": 100},
    ...
  ]
}
```

### ✅ 预期结果

Gemini 应该输出类似这样的 JSON：

```json
{
  "data": [
    {"Region": "华东区", "Month": "7月", "Sales": 100},
    {"Region": "华东区", "Month": "8月", "Sales": 60},
    {"Region": "华东区", "Month": "9月", "Sales": 120},
    {"Region": "华南区", "Month": "7月", "Sales": 40},
    {"Region": "华南区", "Month": "8月", "Sales": 45},
    {"Region": "华南区", "Month": "9月", "Sales": 80},
    {"Region": "华北区", "Month": "7月", "Sales": 50},
    {"Region": "华北区", "Month": "8月", "Sales": 50},
    {"Region": "华北区", "Month": "9月", "Sales": 50}
  ]
}
```

### 💬 讲解要点（对观众说）

1. **强调**："看，Gemini 自动帮我们算出了华东区的销售额（500台×2000元/10000=100万）"
2. **强调**："这个 JSON 数据就是我们要的'金条'，复制保存它！"
3. **对比**："传统方式需要手动打开Excel，建表格，输入公式，至少20分钟"

## 任务二：Trae 搭建基础框架

### 🎯 演示目标
展示 Trae 如何用自然语言快速生成 HTML PPT 框架

### 📋 步骤 1：打开 Trae

1. 访问 [Trae.ai](https://trae.ai)
2. 点击 "New Project"
3. 界面应该是空白的代码编辑器

### 📋 步骤 2：关键 Prompt

**复制粘贴到 Trae：**

```text
请帮我创建一个基于 Reveal.js 的演示文稿（HTML PPT）。

风格要求：
1. 背景使用深灰色 (#2b2b2b)，文字使用白色
2. 引入 ECharts 图表库（CDN链接），因为我后面要用来展示销售数据
3. 页面切换使用 "slide" 过渡效果
4. 添加底部进度条

结构要求：
- 生成 index.html 文件
- 包含 Reveal.js 的基础配置
- 先创建一个封面页，标题是"销售数据分析报告"

请开始编写代码。
```

### ✅ 预期结果

Trae 会生成完整的 HTML 文件，包含：
- Reveal.js 引用
- ECharts CDN
- 基础样式
- 封面页

### 📋 步骤 3：预览效果

1. 点击 Trae 的 "Preview" 按钮
2. 应该看到一个黑色背景的 PPT 封面

### 💬 讲解要点（对观众说）

1. **强调**："看，不到1分钟，我们就有了一个专业的PPT框架"
2. **强调**："我没有写一行代码，只是用自然语言描述需求"
3. **展示**：点开代码，让观众看到 Trae 自动生成的完整 HTML 结构

## 任务三：Trae 生成数据图表

### 🎯 演示目标
将 Gemini 清洗的 JSON 数据注入 Trae，自动生成交互式图表

### 📋 步骤 1：准备 JSON 数据

**复制之前 Gemini 生成的 JSON**（任务一的输出结果）

### 📋 步骤 2：关键 Prompt

**复制粘贴到 Trae：**

```text
这是我的业务数据（JSON格式）：

[粘贴 Gemini 生成的 JSON 数据]

请在 PPT 的第二页，生成一个"分组柱状图"来展示这三个大区的销售对比。

要求：
1. 使用 ECharts 渲染
2. 柱子要有渐变色效果（华东区用蓝色，华南区用紫色，华北区用橙色）
3. 柱子加载要有动态动画（从下往上长出来）
4. 鼠标悬停时显示具体金额（万元）
5. 图例可点击切换显示/隐藏数据
6. 图表标题："Q3 三大区销售对比"

请更新代码。
```

### ✅ 预期结果

Trae 会：
1. 自动解析 JSON 数据
2. 生成完整的 ECharts 配置代码
3. 添加到第二页幻灯片
4. 实现所有交互效果

### 📋 步骤 3：实时预览

1. 保存后自动刷新预览
2. 切换到第二页，应该看到动态图表
3. **现场演示交互**：
   - 鼠标悬停看tooltip
   - 点击图例隐藏/显示数据
   - 切换页面看动画效果

### 💬 讲解要点（对观众说）

1. **强调**："数据自动读取，图表自动生成，我们只花了30秒"
2. **互动**："大家看，鼠标悬停就能看到详细数据，这是静态PPT做不到的"
3. **对比**："传统方式：Excel做图表 → 截图 → 粘贴到PPT → 调整大小，至少15分钟"
4. **高潮**："如果数据变了呢？只需要更新JSON，图表自动刷新！"

## 任务四：Trae 全局样式修改

### 🎯 演示目标
展示一键修改整个PPT风格的能力（模拟老板改需求）

### 📋 场景设定（对观众说）

```
假设老板看完说："这颜色太素了，我要那种'赛博朋克'的感觉，
标题要发光，整体要更有科技感。"

传统方式：你得打开PPT，一页一页改颜色，改标题样式，至少30分钟。

AI方式：看我的...
```

### 📋 关键 Prompt

**复制粘贴到 Trae：**

```text
全局修改演示文稿样式：

1. 背景色改为更深的黑色 (#1a1a1a)
2. 所有标题（h1, h2）改为"霓虹蓝" (#00f3ff)，并添加发光效果（text-shadow: 0 0 20px #00f3ff）
3. 把图表的配色方案改为"蓝紫渐变"（从 #00f3ff 到 #a855f7）
4. 在页面底部增加一个自动播放的进度条（宽度随页面切换增加）
5. 所有文字改为白色 (#ffffff)

请更新全部代码，保持原有功能不变。
```

### ✅ 预期结果

Trae 会：
1. 修改全局 CSS 样式
2. 更新 ECharts 配色
3. 添加进度条HTML和JS
4. 应用霓虹发光效果

### 📋 实时演示效果

1. 保存后自动刷新
2. **展示变化**：
   - 标题发光效果
   - 图表颜色变化
   - 底部进度条
3. **切换页面**：展示进度条动态增长

### 💬 讲解要点（对观众说）

1. **震撼点**："看，整个PPT瞬间变身！我们只花了30秒！"
2. **强调**："Trae 自动修改了所有相关代码，包括CSS、ECharts配置、JS逻辑"
3. **对比**："传统方式手动改，容易漏改，还可能改出bug，至少30分钟起"
4. **总结**："这就是 AI 工作流的威力：数据驱动 + 自然语言控制"

## 🎬 演示流程总结

### 完整时间线（15分钟）

| 时间 | 任务 | 关键动作 | 对观众说的话 |
|------|------|----------|------------|
| 0-5分钟 | Gemini清洗 | 粘贴素材→Prompt→得到JSON | "这是数据大脑在工作" |
| 5-8分钟 | Trae框架 | Prompt→生成HTML→预览 | "3D打印机开始打印" |
| 8-11分钟 | Trae图表 | 注入JSON→生成图表→交互演示 | "数据自己会说话" |
| 11-15分钟 | 样式修改 | Prompt→全局修改→展示变化 | "老板需求秒级响应" |

### 演示技巧 Tips

1. **提前准备**：
   - 素材提前保存在记事本
   - Gemini 和 Trae 都提前登录好
   - 准备一个备用的完整 HTML（万一现场网络问题）

2. **增加互动**：
   - 演示前问观众："你们做PPT通常花多久？"
   - 每完成一步问："大家觉得这个效果怎么样？"
   - 最后问："想试试的举手？"

3. **Meta 玩法（高级技巧）**：
   - 开场时，直接用浏览器打开你做好的 HTML PPT 进行演讲
   - 讲到"实操环节"时，说："其实你们刚才看的这个PPT，就是我用这套方法花10分钟做的"
   - **这种"所见即所得"的震撼感最强！**

4. **容错准备**：
   - 如果 Trae 卡住：说"AI有点害羞，我直接粘贴它之前生成的代码给大家看"
   - 如果网络断了：打开本地准备好的 HTML 文件
   - 如果 Gemini 输出格式不对：准备一个标准 JSON 备用
