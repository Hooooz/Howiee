## 📋 常见问题

### Q1: HTML 文件中文显示乱码

**原因**: 浏览器编码设置问题

**解决**:
1. 右键页面 → 编码 → UTF-8
2. 或重新用现代浏览器打开 (Chrome/Edge/Safari)

---

### Q2: 打印时样式丢失

**原因**: "背景图形" 选项未勾选

**解决**:
1. 打印设置中勾选 "背景图形"
2. 或在 CSS 中添加 `-webkit-print-color-adjust: exact;`

---

### Q3: PDF 文件太大

**原因**: 图片或字体嵌入问题

**解决**:
1. 使用在线压缩工具: https://www.ilovepdf.com/compress_pdf
2. 或使用 Acrobat 优化 PDF

---

### Q4: 某些 Emoji 显示为方框

**原因**: 系统字体不支持某些 Emoji

**解决**:
1. 使用支持 Emoji 的现代浏览器 (Chrome/Edge)
2. 或手动替换为文字描述

---

### Q5: 想要修改样式

**解决**:

1. 打开 `课前手册-PDF版本.html`
2. 找到 `<style>` 标签
3. 修改相应的 CSS 属性

**常用修改**:

```css
/* 修改主标题颜色 */
h1 {
    color: #ff0000;  /* 改为红色 */
}

/* 修改字体大小 */
body {
    font-size: 12pt;  /* 从 10pt 改为 12pt */
}

/* 修改页边距 */
@page {
    margin: 2cm;  /* 从 1.5cm 改为 2cm */
}
```

## 🚀 批量处理 (可选)

如果需要同时生成多个文档的 PDF:

### 使用脚本批量转换

创建文件 `batch-convert.sh`:

```bash
#!/bin/bash

for file in *.md; do
    echo "转换: $file"
    # 调用转换脚本
    python3 convert-md.py "$file"
done

echo "批量转换完成"
```
