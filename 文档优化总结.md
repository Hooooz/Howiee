### 二、分享大纲优化

#### 更新点1：主题与学员收获

**原版：**
```
时长：60分钟
核心隐喻：建筑师 (Gemini) + 3D打印机 (Trae)

- 掌握多维度数据分析（完成率/同比/环比/数据看板）
- 让 AI 自动生成业绩洞察和行动建议
- 效率提升12倍，数据变化自动更新
```

**A部分 - Gemini 核心动作：**
- ❌ 删除：JSON 格式说明

- ✅ 新增：进阶提示（Gemini 可直接计算完成率、同比环比）

**示例对比：**

```

**优化版：**

```

## 📝 使用建议

### 对于讲师

**课前准备：**
1. 熟读优化后的《分享大纲.md》
2. 准备好所有备用素材（CSV、HTML、Prompt卡片）
3. 决定是否现场演示进阶内容（根据学员背景）

**课中策略：**
- **基础学员**：严格按30分钟基础流程，进阶内容一带而过
- **技术背景学员**：可以现场演示完成率分析图表
- **有人提问多维度分析**：引导到课前手册3.3节

**话术模板：**
```
"今天课程重点是掌握基础工作流，完成率、同比环比、数据看板
这些进阶内容在课前手册第3.3节有详细讲解，包含完整的Prompt模板，
建议大家先把基础流程用熟，再深入学习进阶部分。"
```

### 对于学员

**学习路径：**
1. **课前**：阅读课前手册前两步（账号+数据准备）
2. **课中**：跟随讲师完成基础4个任务
3. **课后**：深入学习3.3节进阶分析能力

**能力进阶：**
- **第1周**：熟练掌握基础流程（CSV → 图表）
- **第2周**：尝试完成率分析
