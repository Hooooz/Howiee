
这是一个非常棒的行业切入点！**欧派家居（Oppein）** 的业务场景非常适合展示 AI 的威力：涉及**整装套餐（29800/39800套餐）、多品类（橱柜/衣柜/木门）、复杂的客户交付链条（设计-生产-安装）**。

数据往往散落在门店导购的微信群、安装师傅的反馈表和竞品的促销单页里。

以下是为您定制的**“欧派家居·华南区季度复盘”**实操案例包。

---

### **案例背景故事（讲课时念这段）**

> “各位欧派的伙伴们，场景代入一下：
> 你是华南大区的运营负责人。双11刚过，总部急着要一份**《整装大家居季度复盘报告》**。
> 你现在头很大，因为数据都在群里：
>
> 1. **销售群**里全是导购发的战报（乱七八糟的文本）。
> 2. **售后群**里全是安装师傅和客户的吐槽（关于板材、工期）。
> 3. **市场部**发来几张竞品（索菲亚/尚品宅配）的促销海报照片。
>
> 按照老办法，整理这些要一天。今天我们用 AI，**10分钟**把这些‘碎砖头’变成一座‘精美的样板间’。”

### **第一阶段：Gemini —— 首席设计师（数据清洗）**

**目标：** 把门店群聊和售后反馈，清洗成标准化的 JSON 数据包。

#### **1. 准备“脏数据” (Raw Data)**

*(请提前将以下内容保存在记事本，演示时一次性复制投喂)*

```text
【来源1：华南区门店销售汇报群】
广州旗舰店：本季度“29800整装套餐”爆单了，签了 120 单！但是单品橱柜只有 50 单，衣柜 80 单。
深圳店：我们要惨一点，整装套餐签了 90 单，但是高端定制系列（高定）卖得好，有 40 单。
东莞店：整装套餐 60 单，橱柜 30 单，衣柜 40 单。总体感觉客户对价格很敏感。

【来源2：400客服投诉摘要】
1. 这里的安装师傅不穿鞋套，搞脏了地板，这个月有 15 起投诉。
2. 很多客户反映“爱格板”缺货，发货延迟了 20 天，导致客户很不满，相关投诉 45 起。
3. 设计师出的图纸尺寸不对，现场安装翻车，有 8 起。

【来源3：竞品动态】
索菲亚最近在推“19800 搞定全屋”，还送床垫，对我们 29800 套餐冲击很大。尚品宅配在推“随心选”，主打年轻化设计。
```

#### **2. 发送给 Gemini 的指令 (Prompt)**

*(复制这段指令)*

```text
你是一位欧派家居的大区运营总监。请根据上述杂乱信息，完成以下任务：

1. **经营洞察**：用简练的语言总结我们本季度的核心亮点与风险（100字以内）。
2. **数据结构化（关键）**：请提取数据并整理成纯净的 JSON 格式（不要用Markdown包裹），包含以下两个字段：
   - `sales_data`：统计三个城市（广州、深圳、东莞）的“整装套餐”销量。
   - `complaint_issues`：统计投诉的三大主要原因及数量。
```

#### **3. 预期 Gemini 输出 (你的“数据胶囊”)**

*(如果现场演示不稳定，直接复制这段备用)*

```json
{
  "insight": "本季度整装套餐仍是主力，广州表现最优，但深圳高定有突破。核心风险在于供应链（爱格板缺货）及交付服务（安装不规范），同时面临竞品低价截流的激烈竞争，需强化交付体验与套餐差异化。",
  "sales_data": [
    {"city": "广州旗舰店", "package_sales": 120},
    {"city": "深圳店", "package_sales": 90},
    {"city": "东莞店", "package_sales": 60}
  ],
  "complaint_issues": [
    {"issue": "板材缺货延期", "count": 45},
    {"issue": "安装不规范", "count": 15},
    {"issue": "设计尺寸错误", "count": 8}
  ]
}
```

### **第二阶段：Trae —— 智能样板间搭建（构建 PPT）**

**目标：** 用欧派的品牌色（橙色/金色），生成高大上的汇报页面。

#### **步骤 1：搭建骨架 (Foundation)**

* **操作：** 打开 Trae -> New Project。
* **输入指令：**
  ```text
  创建一个基于 Reveal.js 的演示文稿。
  风格要求：
  1. 欧派品牌高端风：背景使用深灰色 (#333333)，强调色使用“欧派橙” (#ff6600) 和香槟金。
  2. 引入 ECharts 库。
  3. 生成 index.html 和 style.css。
  ```
* **效果：** 预览窗口出现一个带有欧派品牌质感的 PPT 封面。

#### **步骤 2：注入数据 (Data Injection)**

* **操作：** 把 Gemini 生成的 JSON 粘贴进去。
* **输入指令：**
  ```text
  这是我的业务数据（JSON）。请在 PPT 中生成两页图表：

  第一页：标题“整装套餐战报”。使用 ECharts 生成一个“柱状图”，展示三个城市的销量。柱子要用“欧派橙”渐变色。
  第二页：标题“交付痛点分析”。使用 ECharts 生成一个“环形图（Donut Chart）”展示投诉原因。

  要求：图表要有入场动画，看起来要像大厂的数据大屏。
  ```

#### **步骤 3：精细化装修 (Refinement)**

* **场景：** “这页总结太干了，我要加个竞品分析。”
* **输入指令：**
  ```text
  增加最后一页，标题为“核心洞察与对策”。
  内容直接显示 JSON 里的 insight 文本。
  并且在文本下方加一个红色的警告框，写上：“警惕索菲亚 19800 低价策略冲击”。
  ```
